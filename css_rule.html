[ first-child ]

เป็น CSS Selector ที่ใช้ในการเลือกลูกตำแหน่งแรกของตัว Element นั้นๆ

<div>
 <p>Hello World</p>
 <p>Hey this is two line</p>
</div>

จาก html ด้านบนถ้าเราต้องการเลือก p ตัวที่แรกเราก็จะใช้คำสั่งดังต่อไปนี้ได้เลย

div p:first-child { }

[ nth-child() ]

เป็น CSS Selector ที่ใช้ในการเลือกลูกตำแหน่งที่ n ของตัว Element นั้นๆ

<div>
 <p>Hello World</p>
 <p>Hey this is two line</p>
</div>

จาก html ด้านบนถ้าเราต้องการเลือก p ตัวที่สองเราก็จะใช้คำสั่งดังต่อไปนี้ได้เลย

div p:nth-child(2) { }

[ transition ]

เป็น Property ที่จะทำการทำให้การเปลี่ยน Property อื่นๆ เกิดขึ้นอย่างช้าๆ เป็น Animation หรือเป็นการเพิ่ม transition ให้ตัว Element นั่นเอง โดยการใช้งานจะต้องใช้งานเป็นดังนี้ 

transition: property-name time speed-curve

อธิบาย Parameters ต่างๆ ดังนี้

  • Property-name : คือชื่อของ Property ที่เราต้องการใส่ Transition ลงไป
  • Time : คือระยะเวลาว่าจะให้เกิด Transition นานแค่ไหน
  • Speed-Curve : คือ Preset ที่จะเป็นเหมือนรูปแบบการเล่นของอนิเมชั่น เช่น
    • ease : เริ่มต้นช้ากลางเร็วและปลายช้า
    • linear : ไม่มีการเปลี่ยนแปลงความเร็ว
    • ease-in : เริ่มต้นช้า
    • ease-out : จบช้า
    • ease-in-out : ช้าในตอนเริ่มต้นและตอนจบ

Example of use

transition: background-color 0.5s ease-in;
Hover Me without transition :( Hover Me with transition :)

[ cursor ]

ใช้เพื่อทำการปรับเปลี่ยนรูปแบบของ mouse เมื่อไป Hover ที่ element นั้นๆ

cursor: value;

โดยตัว Value คือ รูปแบบเมาส์ที่จะเปลี่ยนไปซึ่งจะมีเยอะมากๆ ดังนั้นเราขอแนบเป็นลิ้งไว้แทนการเขียนทั้งหมด Ref

Example of use

cursor: all-scroll;
Hello World

[ transform ]

ใช้ในการเปลี่ยนแปลง element นั้นๆ ทั้งในระบบ 2 มิติ และ 3 มิติ เช่น การหมุน Scale

transform: none|transform-functions|initial|inherit;

โดยตัว transform-functions มีหลายตัวมากๆ ดังนั้นเราขอแนบเป็นลิ้งไว้แทนการเขียนทั้งหมด Ref

Example of use

transform: rotation(45deg)
Ops Rotated
